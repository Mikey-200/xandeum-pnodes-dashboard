# Xandeum pNodes Explorer

Live analytics dashboard for Xandeum storage provider nodes (pNodes).

## Features
- Real-time pNode list fetched via official pRPC
- Sortable & searchable table
- Auto-refresh every 60 seconds
- Fully responsive + dark mode
- Deployed on Vercel

## Live Demo
https://your-app.vercel.app

## Tech Stack
Next.js 14 • TypeScript • Tailwind • TanStack Table • TanStack Query
